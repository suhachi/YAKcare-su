## ⚠️ 중요 사항

### 1. `.env.local` 파일은 Git에 커밋하지 마세요!

이 파일에는 **민감한 정보**가 포함되어 있습니다. 
다음 파일이 `.gitignore`에 포함되어 있는지 확인하세요:

```gitignore
# .gitignore
.env.local
.env*.local
```

### 2. 개발 서버 재시작 필요

환경 변수를 변경한 후에는 **반드시 개발 서버를 재시작**해야 합니다:

```bash
# 현재 실행 중인 서버 중지 (Ctrl+C)
# 그 다음 재시작
npm run dev
```

### 3. 브라우저 캐시 클리어

변경사항이 반영되지 않으면:
1. 브라우저 **강력 새로고침** (Ctrl+Shift+R)
2. 또는 **Application > Clear storage** 후 새로고침

## ✅ 확인 사항

다음 오류가 사라졌는지 확인하세요:

- ❌ `[Supabase Client] ❌ 필수 환경 변수가 누락되었습니다`
- ❌ `Uncaught Error: Supabase 환경 변수 누락`

콘솔에 다음 메시지가 표시되어야 합니다:

- ✅ `[Supabase Client] ✅ Initialized`
- ✅ `[Supabase Client] Supabase URL: ✓ Set`
